Exercise 1.2. Formula Type (*)

Question:

Determine which of the previous formulas are tautologies, contradictions, or non-tautological satisfiable formulas. Explain your answer.

Solution

Based on the truth tables above:

  1. Satisfiable but non-tautological: It is only True when \(x\), \(y\), and \(z\) are all False.
  2. Contradiction: It evaluates to False under all possible truth assignments.
  3. Tautology: It evaluates to True under all possible truth assignments.
  4. Satisfiable but non-tautological: It is False only when \(x\), \(y\), and \(z\) are all True, and True otherwise.
  5. Satisfiable but non-tautological: It yields True when \(x\) is False and False when \(x\) is True.
  6. Satisfiable but non-tautological: It contains a mix of True and False evaluations (e.g., True for F, F, F, T, but False for T, T, F, T).

Exercise 1.4. Formula Construction (**)

Question:

Build (i) a tautology and (ii) a contradiction which use at least three propositional variables.

Solution

We can explicitly guarantee the use of at least three propositional variables (let's use \(x, y,\) and \(z\)) like so:

i) Tautology:
\((x \land y \land z) \lor \neg(x \land y \land z)\)
(This evaluates to True regardless of the variables' states).

ii) Contradiction:
\((x \land y \land z) \land \neg(x \land y \land z)\)
(This evaluates to False regardless of the variables' states).

Exercise 1.5. Formula Construction I (*)

Question:

Build a formula over four propositional variables which is true if and only if any two of the input variables is true.

Solution

Let the four propositional variables be \(a, b, c,\) and \(d\). The phrasing "if any two [...] is true" typically distinguishes between two subtle meanings in logic. Here is the formula based on the most direct interpretations:

Interpretation A: "At least two variables are true" (Most standard logical translation for 'if any two...'):
You just require finding any two un-negated variables combined by an AND.

\(f(a,b,c,d) = (a \land b) \lor (a \land c) \lor (a \land d) \lor (b \land c) \lor (b \land d) \lor (c \land d)\)

Interpretation B: "Exactly two variables are true" (In case it is contrasting with 'exactly one' from Ex 1.3):

\(f(a,b,c,d) = (a \land b \land \neg c \land \neg d) \lor (a \land \neg b \land c \land \neg d) \lor (a \land \neg b \land \neg c \land d) \lor (\neg a \land b \land c \land \neg d) \lor (\neg a \land b \land \neg c \land d) \lor (\neg a \land \neg b \land c \land d)\)

Exercise 1.6. Formula Construction II (**)

Build a formula that represents the Boolean function expressed by the following table:

\(x\) \(y\) \(z\) \(\varphi\)
0001
0010
0101
0111
1000
1011
1100
1111

Solution

  1. Identify the minterms (rows where \(\varphi = 1\)):
    • Row 1: \(x=0, y=0, z=0 \implies (\neg x \land \neg y \land \neg z)\)
    • Row 3: \(x=0, y=1, z=0 \implies (\neg x \land y \land \neg z)\)
    • Row 4: \(x=0, y=1, z=1 \implies (\neg x \land y \land z)\)
    • Row 6: \(x=1, y=0, z=1 \implies (x \land \neg y \land z)\)
    • Row 8: \(x=1, y=1, z=1 \implies (x \land y \land z)\)
  2. Combine to form the Disjunctive Normal Form (DNF):
    \(\varphi = (\neg x \land \neg y \land \neg z) \lor (\neg x \land y \land \neg z) \lor (\neg x \land y \land z) \lor (x \land \neg y \land z) \lor (x \land y \land z)\)
  3. Simplify the formula (grouping terms):
    • Combine rows 1 and 3: \((\neg x \land \neg y \land \neg z) \lor (\neg x \land y \land \neg z) \equiv \neg x \land \neg z\)
    • Combine rows 4 and 8: \((\neg x \land y \land z) \lor (x \land y \land z) \equiv y \land z\)
    • Combine rows 6 and 8: \((x \land \neg y \land z) \lor (x \land y \land z) \equiv x \land z\)

    Putting it together yields the simplified formula:

    \(\varphi = (\neg x \land \neg z) \lor (y \land z) \lor (x \land z)\)

    (Note: This can be further factored to \(\varphi = (\neg x \land \neg z) \lor (z \land (x \lor y))\) )

Exercise 1.7. Formula Properties (**)

Let \(\varphi\) and \(\psi\) be two propositional formulas. Check which of the following statements are true, providing a justification:

  1. if \(\varphi\) and \(\psi\) are satisfiable, then \(\varphi \land \psi\) is satisfiable;
  2. if \(\varphi\) and \(\psi\) are satisfiable, then \(\varphi \lor \psi\) is satisfiable;
  3. if \(\varphi\) and \(\psi\) are satisfiable, then \(\varphi \land \neg\psi\) is satisfiable;
  4. if \(\varphi\) and \(\psi\) are tautologies, then \(\varphi \land \psi\) is a tautology;
  5. if \(\varphi\) and \(\psi\) are contradictions, then \(\varphi \lor \psi\) is a contradiction;
  6. if \(\varphi\) is a tautology, then \(\varphi \to \psi\) is satisfiable;
  7. if \(\varphi\) is a tautology, then \(\psi \to \varphi\) is a tautology.

Solution

  1. if \(\varphi\) and \(\psi\) are satisfiable, then \(\varphi \land \psi\) is satisfiable;
    False. Justification: Counterexample: Let \(\varphi = p\) and \(\psi = \neg p\). Both are independently satisfiable, but \(\varphi \land \psi \equiv p \land \neg p\), which is a contradiction (never satisfiable).

  2. if \(\varphi\) and \(\psi\) are satisfiable, then \(\varphi \lor \psi\) is satisfiable;
    True. Justification: If \(\varphi\) is satisfiable, there is at least one truth assignment that makes it True. Under this same assignment, the disjunction \(\varphi \lor \psi\) will evaluate to True, making it satisfiable.

  3. if \(\varphi\) and \(\psi\) are satisfiable, then \(\varphi \land \neg\psi\) is satisfiable;
    False. Justification: Counterexample: Let \(\varphi = p\) and \(\psi = p\). Both are satisfiable, but \(\varphi \land \neg\psi \equiv p \land \neg p\), which is a contradiction.

  4. if \(\varphi\) and \(\psi\) are tautologies, then \(\varphi \land \psi\) is a tautology;
    True. Justification: A tautology is True under all truth assignments. Since \(\varphi\) and \(\psi\) are both always True, \(\varphi \land \psi\) will evaluate to \(\text{True} \land \text{True} \equiv \text{True}\) under all assignments.

  5. if \(\varphi\) and \(\psi\) are contradictions, then \(\varphi \lor \psi\) is a contradiction;
    True. Justification: A contradiction is False under all assignments. Because \(\varphi\) and \(\psi\) are always False, \(\varphi \lor \psi\) evaluates to \(\text{False} \lor \text{False} \equiv \text{False}\) under all assignments.

  6. if \(\varphi\) is a tautology, then \(\varphi \to \psi\) is satisfiable;
    False. Justification: Counterexample: Let \(\psi\) be a contradiction (e.g., \(q \land \neg q\)). Because \(\varphi\) is a tautology (always True), the implication \(\varphi \to \psi\) evaluates to \(\text{True} \to \text{False}\), which equals False under all assignments, making it unsatisfiable.

  7. if \(\varphi\) is a tautology, then \(\psi \to \varphi\) is a tautology.
    True. Justification: By definition, an implication \(A \to B\) is equivalent to \(\neg A \lor B\). Here, it translates to \(\neg\psi \lor \varphi\). Since \(\varphi\) is a tautology (always True), the disjunction \(\neg\psi \lor \text{True}\) is always True, making the entire expression a tautology.

Exercise 1.8. Variable Accumulation (**)

Verify that:

\(x \land y \equiv x \land y \land (w \lor \neg w)\)

Solution

  1. According to the Law of Excluded Middle, a variable or its negation (\(w \lor \neg w\)) is always True (it is a tautology).
  2. Therefore, substitute \((w \lor \neg w)\) with \(\top\) (True):
    \(x \land y \land (w \lor \neg w) \equiv x \land y \land \top\)
  3. According to the Identity Law of Conjunction, \(A \land \top \equiv A\).
  4. Applying this to our formula:
    \(x \land y \land \top \equiv x \land y\).
    Thus, the equivalence \(x \land y \equiv x \land y \land (w \lor \neg w)\) is fully verified.

Exercise 1.9. Shefter (***)

  1. Describe the NAND operator, which is true if at least one of its parameters is false as a truth table and as a formula.
  2. Show how one can express conjunction (\(\land\)), disjunction (\(\lor\)), and negation (\(\neg\)) using only NAND.

Solution

(Note: "Shefter" is a typo in the provided document; it refers to the Sheffer stroke / NAND gate).

1. Describe the NAND operator as a truth table and as a formula.

Formula: \(p \text{ NAND } q \equiv \neg(p \land q)\)

Truth Table:

\(p\) \(q\) \(p \text{ NAND } q\)
001
011
101
110

2. Show how one can express conjunction (\(\land\)), disjunction (\(\lor\)), and negation (\(\neg\)) using only NAND.

  • Negation (\(\neg p\)):
    We need an output that is opposite to \(p\).
    \(\neg p \equiv p \text{ NAND } p\)
    (Proof: \(\neg(p \land p) \equiv \neg p\))

  • Conjunction (\(p \land q\)):
    We know \(p \land q\) is the exact inverse of \(p \text{ NAND } q\). Thus, we simply negate a NAND using the negation setup shown above.
    \(p \land q \equiv \neg(p \text{ NAND } q)\)
    \(p \land q \equiv (p \text{ NAND } q) \text{ NAND } (p \text{ NAND } q)\)

  • Disjunction (\(p \lor q\)):
    According to De Morgan's laws, \(p \lor q \equiv \neg(\neg p \land \neg q)\). This matches the definition of a NAND operation on inverted inputs (\((\neg p) \text{ NAND } (\neg q)\)).
    Substitute the negation rule for both \(p\) and \(q\):
    \(p \lor q \equiv (p \text{ NAND } p) \text{ NAND } (q \text{ NAND } q)\)
